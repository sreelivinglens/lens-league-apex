⚠ Danger Zone
Destructive actions — use with caution ▼
Rescore Images with Missing Data
Re-runs scoring only on images where audit data is incomplete or missing — not a calibration tool. Does not touch photographs already evaluated with complete data. Scores are permanent records of performance at the time of submission; use Force Rescore on individual images only when a specific error is identified (wrong genre, corrupted EXIF, false AI flag). Uses API credits and may take several minutes.
Refresh AI Suspicion Scores
Re-runs AI detection only on all photographs evaluated. Does NOT change scores, tiers, or any other data. Run this after updating the detection prompt to backfill existing images. Images scoring ≥0.7 are automatically moved to needs-review.
Fix Beta Plan Labels
Updates all manually-activated beta users to monthly plan so MRR calculates correctly.
Reset All Peer Rating Credits & Data
Resets all users' rating_credits to 0 and clears all peer rating data — assignments, ratings, pool entries, blended scores. Cannot be undone.
